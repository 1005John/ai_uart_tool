# 变更日志

## [v1.2.1] - 2026-05-31

### 新增
- 缺陷报告自动附带完整执行日志（>>>发送 / <<<接收）
- 手动创建缺陷功能：自定义测试环境/实际结果/失败原因 + 完整日志
- 执行日志自动保存到 data/last_exec_log.json

## [v1.2.0] - 2026-05-31

### 重构
- **UI 全面重写**：双标签页 + 表单驱动
  - 📦 用例集库：左侧 Radio 列表 + 右侧 Dataframe 可编辑表格（点击行选中 + 按钮操作）
    - 用例集：新建 / 复制 / 删除（按钮横排）
    - 用例：新增空行直接填写 / 删除选中 / 上移下移 / 保存
  - 📋 方案管理：左中右三列布局
    - 左：方案列表 + 执行轮次/间隔 + 用例集勾选（勾选即保存）
    - 中：Dataframe 用例表格（勾选→执行）+ 状态显示
    - 右：AT 指令日志框（>>>发送 / <<<接收实时打印）
  - 🐛 缺陷管理：左右布局（左：列表+灵畿提交 / 右：缺陷详情）+ Textbox 日期筛选
- 数据模型：用例集独立维护编辑 → 方案勾选组合为只读视图
- 方案执行：支持多轮循环 + 轮次间隔延迟

### 新增
