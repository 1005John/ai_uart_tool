# AI Native UART Tool — 需求规格说明

> 版本：v1.4 | 日期：2026-06-03 | 状态：已确认

## 1. 项目背景

物联网蜂窝模组的 AT 指令测试长期依赖手工逐条发送和人工判读，效率低、易遗漏。
本项目为测试工程师提供一套 AI 驱动的自动化测试工具，覆盖从用例加载、执行、结果判定到缺陷提交的完整闭环。

目标用户：物联网模组测试工程师（中移物联）。
运行环境：树莓派 5 / Ubuntu / Debian / macOS。

## 4. 约束与假设

- 运行环境：树莓派 5 / Ubuntu / Debian / macOS，Python 3.10+
- 模组通过 USB 串口连接（vendor=0x3563 product=0x2000）
- 测试用例以 Excel 文件形式提供（test_data/ 目录下 23+ 个 xlsx）
- DeepSeek API 可正常访问
- 灵畿 CLI（lc）已安装并登录
